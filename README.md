# Web Experiments

A collection of small experiments and prototypes built with HTML, CSS, and JavaScript.  
This serves as a personal playground for testing visual ideas, interaction patterns, and creative coding techniques. 

## How to Run

Clone the repository and open any experiment folder.
Each project runs independently in the browser.
       
## Technologies

- HTML
- CSS
- JavaScript
- p5.js (some experiments)

## Notes

These projects are exploratory and may be incomplete.  
They are intended as learning exercises and visual prototypes.
